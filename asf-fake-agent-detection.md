# ASF-12: Fake Agent Detection System
**Priority:** HIGH (Based on Shapiro Intelligence)
**Layer:** Community Security (Expansion of Layer 2)

## Problem Statement
David Shapiro reveals 99% of Moltbook's 1.5M "agents" are fake:
- Posts written by humans, run through chatbots, submitted via OpenClaw
- Massive API key leak (1.5M compromised)
- Platform became highway for crypto scams and malware
- "Mockly" tool generates fake Moltbook screenshots

## ASF-12 Solution: Fake Agent Detection Framework

### Detection Methods:
1. **Behavioral Analysis**
   - Post timing patterns (humans vs genuine agents)
   - Response authenticity scoring
   - Engagement pattern recognition

2. **Technical Verification**
   - API key authentication chains
   - Source code transparency requirements
   - Deployment environment verification

3. **Community Validation**
   - Peer review system for agent claims
   - Reputation scoring based on verifiable work
   - Cross-platform consistency checks

### Implementation Tools:
```bash
# fake-agent-detector.sh
# Analyzes posting patterns, API usage, behavioral consistency
# Scores agents: 0-5 (likely fake), 6-10 (authentic), 11-15 (verified real)
```

### Verification Levels:
- **Level 1:** Basic activity pattern analysis
- **Level 2:** Technical verification (code/deployment proof)  
- **Level 3:** Community vouching system
- **Level 4:** Real-world work verification

### Integration with Existing ASF:
- **Layer 0 (Code Security):** skill-evaluator.sh catches malicious agent code
- **Layer 1 (Network Security):** port-scan-detector.sh protects against attacks
- **Layer 2 (Community Security):** spam detection + fake agent detection
- **Layer 3 (Infrastructure Security):** VPN/antivirus baseline

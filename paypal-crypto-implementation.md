## Step 2: Verify Current Plugin (2 minutes)

1. **Go to:** Plugins → Installed Plugins
2. **Find:** PayPal plugin (likely "PayPal Payments" or similar)
3. **Check version:** Should be 2.0+ for crypto support
4. **Update if needed:** Click update if available

2. **Find PayPal option** → Click "Manage"
3. **Look for:** Cryptocurrency, Bitcoin, or Digital Currency options
4. **Enable if present:** Turn on crypto payment methods

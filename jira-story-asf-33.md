# ASF-33: Create Agent Security Framework Website

## Story Details

**Title:** Create ASF website at scrumai.org/agentsecurityframework

**Story Type:** Feature
**Priority:** High
**Sprint:** ASF Sprint 2
**Story Points:** 5
**Assignee:** ASF Sales Agent

## Description

Create a professional website for the Agent Security Framework that showcases the tool, its features, and helps drive adoption. The site should have a modern design similar to Moltbook or OpenClaw.

## Acceptance Criteria

1. [ ] Design homepage with ASF branding and value proposition
2. [ ] Create "Features" section highlighting:
   - Self-healing security scanning
   - Vulnerability detection
   - Auto-fix capabilities
   - Responsible disclosure process
3. [ ] Add "Getting Started" guide for new users
4. [ ] Include download/installation instructions
5. [ ] Create documentation section
6. [ ] Add testimonials/use cases section
7. [ ] Include GitHub repository link
8. [ ] Mobile-responsive design
9. [ ] SEO optimized for "agent security" keywords

## Technical Requirements

- Host at: scrumai.org/agentsecurityframework
- Modern, clean design (similar to Moltbook/OpenClaw)
- Fast loading, static site preferred
- Compatible with existing scrumai.org infrastructure

## Resources

- Existing web team at scrumai.org for support
- Site already has traffic to leverage
- ASF GitHub repo: https://github.com/jeffvsutherland/asf-security-scanner
- ASF documentation in repo

## Success Metrics

- Site live and accessible
- Clear call-to-action for downloads
- Professional appearance that builds trust
- Easy navigation to key information

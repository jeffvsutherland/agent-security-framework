## Definition of Done ‚úÖ
A story is NOT done when you think it's done. It's done when it meets the Definition of Done:
- Code written, reviewed, merged
- All tests pass
- Documentation updated
- Increment is deployable
- **Public deliverables audited by external expert**
- Product Owner accepted

**Never move an incomplete story to Done. This injects entropy.**
